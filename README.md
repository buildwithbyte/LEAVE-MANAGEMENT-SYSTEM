# Employee Leave Management System

## 📖 Project Overview
Employee Leave Management System is a web-based application that automates the leave request and approval process within an organization. Employees can submit leave requests, while managers can review, approve, or reject them. The system also helps track leave balances and employee availability.

## 🚀 Features
- User Authentication (Employee & Manager Roles)
- Leave Request Submission
- Leave Approval/Rejection Workflow
- Leave Balance Management
- Leave Calendar View
- Form Validation
- Database Integration

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- MySQL

## 📂 Modules
### Employee
- Login
- Apply for Leave
- View Leave Status
- Check Leave Balance

### Manager
- Login
- Review Leave Requests
- Approve or Reject Requests
- Manage Employee Leave Records

## 🎯 Purpose
The goal of this project is to simplify leave management, reduce paperwork, and improve transparency between employees and managers.
